# **App Name**: Hagaaty AI Blog

## Core Features:

- AI-Powered Article Generation: Generate blog articles using the Open Router API (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) as a tool. The tool should decide which elements to include in each article based on user preference, subject matter, etc.
- Smart Assistant Chat Interface: Integrate a chat interface powered by the Open Router API (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) to answer user questions and provide helpful information.
- Article Categorization and Tagging: Automatically categorize and tag articles using AI to improve searchability and discoverability.
- Blog Post Display: Display blog posts in a clear and engaging layout with options for sorting and filtering.
- User Authentication: Allow users to create accounts and log in to save their favorite articles and settings.
- Search Functionality: Implement a search bar to quickly find relevant articles by keywords.

## Style Guidelines:

- Primary color: Deep Indigo (#4B0082) to represent intelligence and innovation.
- Background color: Very light lavender (#F0F8FF) to create a clean and readable backdrop.
- Accent color: Teal (#008080) to highlight key elements and calls to action.
- Body and headline font: 'Inter' sans-serif for a modern, neutral look.
- Use clean and modern icons to represent various categories and functions.
- Maintain a clean and organized layout to ensure readability and ease of navigation.
- Incorporate subtle animations for loading states and transitions to enhance the user experience.
